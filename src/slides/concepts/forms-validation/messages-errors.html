Fehler-Meldungen
  • Detailierte Meldungen pro Validator möglich
    • errors-Map mit Informationen pro Validator
  • Code wird schnell unübersichtlich
    • Bibliothek mit Komponenten für Fehlermeldungen verwenden
<form [formGroup]="form">
  <input type="password" formControlName="password">
  <div [hidden]="!form.controls.password.dirty">
    <div [hidden]="!form.controls.password.errors?.required">
      Password is required
    </div>
    <div [hidden]="!form.controls.password.errors?.complex">
      Password is not complex enough
    </div>
  </div>
</form>
